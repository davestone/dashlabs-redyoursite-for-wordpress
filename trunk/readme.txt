== Description ==

The redyoursite.com for WordPress plugin automatically adds the redyoursite.com code to your blog.

== Installation ==

This section describes how to install the plugin and get it working.

1. Delete any existing `dashlabs-redyoursite` folder from the `/wp-content/plugins/` directory
2. Upload `dashlabs-redyoursite` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
